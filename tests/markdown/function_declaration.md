## Test case: simple function
JavaScript:
```js
function Test() {
    var tmp = 10;
}
```

Python:
```py
def Test():
    tmp = 10
```

## Test case: function with multiple lines
JavaScript:
```js
function Test() {
    var tmp = 10;
    tmp += 1
}
```

Python:
```py
def Test():
    tmp = 10
    tmp += 1
```

## Test case: simple function with return
JavaScript:
```js
function Test() {
    return 10;
}
```

Python:
```py
def Test():
    return 10
```

## Test case: function with argument
JavaScript:
```js
function Test(arg1) {
    var tmp = 10;
}
```

Python:
```py
def Test(arg1):
    tmp = 10
```

## Test case: function with multiple arguments
JavaScript:
```js
function Test(arg1, arg2, arg3) {
    var tmp = 10;
}
```

Python:
```py
def Test(arg1, arg2, arg3):
    tmp = 10
```
