## Test case: member expression access property
JavaScript:
```js
hello.world
```

Python:
```py
hello.world
```

## Test case: member expression call function
JavaScript:
```js
hello.world()
```

Python:
```py
hello.world()
```

## Test case: member expression call function with parameter
JavaScript:
```js
hello.world("greet")
```

Python:
```py
hello.world('greet')
```

Python:
